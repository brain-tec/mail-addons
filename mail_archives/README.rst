Mail Archives
=============

Adds Archive menu, which shows sent/received messages

Usage
-----
Click Discuss/Archive menu -- sent/received messages are displayed

Further information
-------------------

Odoo Apps Store: https://apps.odoo.com/apps/modules/9.0/mail_archives/
